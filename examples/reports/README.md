# Example Profiler Reports

This directory contains example HTML reports generated by the DataK9 Profiler.

## Available Examples

### profiler_example_with_semantic_tagging.html

**Dataset**: Financial transactions (100 rows)
**Features Demonstrated**:
- FIBO-based semantic tagging
- Intelligent validation suggestions
- Data quality scoring
- PII detection
- Statistical analysis
- Pattern detection
- Temporal analysis

**View Online**: [Live Example Report](https://github.com/yourusername/data-validation-tool/blob/main/examples/reports/profiler_example_with_semantic_tagging.html)

## How to Generate Your Own

Profile any CSV, Excel, JSON, or Parquet file:

```bash
python3 -m validation_framework.cli profile data.csv -o report.html
```

With semantic tagging enabled:

```bash
python3 -m validation_framework.cli profile data.csv \
  -o report.html \
  --enable-semantic-tagging
```

See [Data Profiling Guide](../../docs/using-datak9/data-profiling.md) for complete documentation.
