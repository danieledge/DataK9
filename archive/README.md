# Archive Directory

This directory contains historical files for reference and debugging purposes.

## Structure

### test-reports/
Generated validation and profiling reports from various test runs:
- HTML reports with interactive visualizations
- JSON summaries with detailed results
- Log files from validation runs

**Note**: These files are git-ignored to keep the repository clean.

### demo-examples/
Example scripts and backup files:
- Demo Python scripts
- Studio HTML backups
- Experimental implementations

**Note**: These files are git-ignored.

### documentation/
Historical documentation files:
- DEMO.md - Old demo instructions
- MONACO_SELF_HOSTED.md - Monaco editor setup
- OFFLINE_MODE.md - Offline usage guide
- RUNNING_TESTS.md - Old test instructions

**Note**: Current documentation is in `docs/` directory.
